  <behavior>
    - HEX round-trip: parse "#FF8000" → RGBA (1.0, 0.5019…, 0.0, 1.0); emit back to "#FF8000".
    - HEX with alpha: "#FF800080" parses alpha ≈ 0.502.
    - RGB↔HSL round-trip: pure red (1,0,0) → HSL(0,1,0.5) → back to (1,0,0).
    - RGB↔HSV round-trip: pure red → HSV(0,1,1) → back. Guard the achromatic S=0 divide-by-zero edge.
    - WCAG ratio: black vs white returns 21.0:1; passes AA normal (≥4.5) and AAA normal (≥7.0).
    - OKLCH: a reference vector from oklch.com / CSS Color 4 (bake 2-3 concrete values) round-trips sRGB→OKLCH→sRGB within tolerance.
    - Out-of-gamut: an OKLCH value known to exceed sRGB returns isOutOfGamut == true with clamped 0...1 channels.
    - Garbage: "#ZZZ" / "" → .failure or graceful nil, never crash.
  </behavior>
  <action>Create `enum ColorTransformer` (`import Foundation` + `import ChromaKit`; NO `import SwiftUI` in the math core). Canonical internal representation is sRGB RGBA `(red, green, blue, alpha)` all in `0.0...1.0`. Implement pure functions: HEX parse/emit (3/6/8 digit, alpha); RGB↔HSL; RGB↔HSV (guard achromatic S=0/V edge for divide-by-zero); `wcagContrastRatio(_ a: RGBA, _ b: RGBA) -> Double` using the W3C formula (linearize each channel `c ≤ 0.03928 ? c/12.92 : ((c+0.055)/1.055)^2.4`; `L = 0.2126R+0.7152G+0.0722B`; `ratio = (Llight+0.05)/(Ldark+0.05)`) and `wcagResults(ratio:) -> (aaNormal, aaLarge, aaaNormal, aaaLarge: Bool)` against thresholds 4.5/3.0/7.0/4.5. OKLCH forward: OKLCH→sRGB via ChromaKit's `NSColor.oklch(l,c,h,alpha)` then `.usingColorSpace(.sRGB)` (handle nil) and read components, AND compute the raw unclamped channels to detect gamut — return `struct GamutResult { let rgba: RGBA; let isOutOfGamut: Bool }` where isOutOfGamut is true if any raw channel ∉ [0,1] (ChromaKit silently clamps; do the range check yourself). OKLCH reverse (sRGB→OKLCH) is hand-computed via ChromaKit's public Core structs (Oklch/Oklab/XYZ) or the CSS-L4 inverse matrix; bake reference vectors to verify. All functions return `Result` or are total (never force-unwrap; INFRA-17). Write `FlintTests/ColorTransformerTests.swift` covering every behavior above with documented reference vectors (oklch.com / WCAG known pairs).</action>
